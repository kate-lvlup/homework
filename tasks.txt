TaskOne
написать класс который выводит в консоль первый елемент аргумментов командной строки(command line argument)

TaskTwo
написать класс который выводит в консоль сумму, разность, произведение, результат деления, остаток от деления и процент первого от второго числа, значение получаем аргументом командной строки

TaskThree
написать класс который выводит в консоль результат конвертации градусов по цельсию в фаренгейты, значение получаем аргументом командной строки

TaskFour
написать класс который выводит в консоль сегодняшнюю дату и время

TaskFive
написать класс который выводит в консоль большее из двух заданных чисел,  значение получаем аргументом командной строки

TaskSix
написать класс который выводит в консоль большее из трех заданных чисел,  значение получаем аргументом командной строки

TaskSeven
написать класс который выводит в консоль 3 рандомно сгенерированных числа больше 0 и меньше 100

TaskEight
написать класс который выводит в  консоль век из полученной даты (например 873 год - это 9й век, 1701 год это 18 век),  значение получаем аргументом командной строки
